# PostureGuard

On-device posture classification for the M5StickS3 using BMI270 motion data and TensorFlow Lite Micro.

**Status:** In progress — hardware incoming.

## Overview

PostureGuard is a wellness and ergonomics prototype focused on edge inference for posture classification. It is a sequel to wearable posture research conducted at Aoyama Gakuin University in Summer 2025.
